# 🍽️ Menu Miraq — نظام إدارة المطاعم المتكامل

## الميزات الرئيسية

### 🔧 لوحة المطوّر (Developer Panel)
- إنشاء مطاعم جديدة مع اسم مستخدم وكلمة مرور
- إدارة الاشتراكات (يومي، شهري، 3 أشهر، 6 أشهر، سنوي)
- تفعيل/إيقاف المطاعم فوراً
- تجديد الاشتراكات
- إحصائيات المطاعم

### 🏪 لوحة الأدمن (Restaurant Admin)
- **تسجيل دخول** باسم المستخدم وكلمة المرور (بدون كود)
- **رابط المنيو العامة** يظهر مباشرة في اللوحة
- **تغيير اسم المطعم** والمعلومات
- **رفع شعار (Logo)** للمطعم
- **تغيير ألوان المنيو** (أساسي، ثانوي، خلفية)
- **تغيير رقم الواتساب** لاستقبال الطلبات
- **تنبيه صوتي** للطلبات الجديدة
- **إدارة الفئات**: إضافة، تعديل، حذف، ترتيب
- **إدارة المنتجات**: إضافة صورة، تفعيل/تعطيل، فلترة حسب الفئة
- **إدارة الطاولات**: إنشاء QR لكل طاولة
- **إدارة الطلبات**: قبول، تحضير، جاهز، تسليم، إلغاء

### 📋 المنيو العامة (Public Menu)
- عرض المنيو حسب الفئات
- إضافة للسلة
- إرسال الطلب مع رقم الهاتف
- **تتبع الطلب** (Tracking) مع خطوات التنفيذ
- **QR للطاولات**: الزبون يمسح QR ويطلب مباشرة

## طريقة التشغيل

```bash
# 1. تثبيت Node.js
# 2. تشغيل السيرفر
node server.js

# 3. فتح لوحة المطوّر
http://localhost:5000/dev
# كلمة المرور الافتراضية: dev@muqtada2025

# 4. إنشاء مطعم جديد من لوحة المطوّر
# 5. فتح لوحة الأدمن
http://localhost:5000/admin
# اسم المستخدم وكلمة المرور اللي أنشأتهم

# 6. عرض المنيو العامة
http://localhost:5000/menu/[slug]
```

## البيانات المحفوظة
- `data/restaurants.json` — بيانات المطاعم
- `data/orders.json` — الطلبات
- `data/tables.json` — الطاولات
- `data/dev-config.json` — إعدادات المطوّر

## المسارات

| المسار | الوصف |
|--------|-------|
| `/dev` | لوحة المطوّر |
| `/admin` | لوحة الأدمن |
| `/menu/:slug` | المنيو العامة |
| `/table/:slug/:number` | طلب من طاولة (QR) |
| `/track/:orderId` | تتبع الطلب |
| `/qr/:slug/:number` | صفحة QR للطاولة |
