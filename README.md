# نظام إدارة اشتراكات المطاعم

## الملفات
- `server.js`      — السيرفر (يعمل عندك أنت)
- `developer.html` — لوحة تحكمك الشخصية
- `index.html`     — موقع المطعم (ترسله للزبون)
- `licenses.json`  — قاعدة البيانات (تُنشأ تلقائياً)

---

## طريقة التشغيل

### 1. تثبيت Node.js
https://nodejs.org (مرة واحدة فقط)

### 2. تشغيل السيرفر
```
node server.js
```
سيعمل على: http://localhost:3000

### 3. فتح لوحة المطوّر
http://localhost:3000/developer.html
كلمة السر الافتراضية: dev@muqtada2025

### 4. إرسال الموقع للزبون
- أرسل ملف index.html للزبون
- عنوان السيرفر في الملف: http://localhost:3000
- ⚠️ للاستخدام الحقيقي غيّر العنوان لسيرفر حقيقي (Hostinger, VPS, إلخ)

---

## كيف يشتغل النظام

1. **توليد اشتراك**: من developer.html → "اشتراك جديد" → انسخ المفتاح وأرسله للزبون
2. **تفعيل عند الزبون**: يدخل المفتاح مرة واحدة → يُحفظ تلقائياً
3. **إيقاف فوري**: اضغط ⏸ في developer.html → الموقع يتوقف فوراً عند كل الزبائن
4. **تشغيل فوري**: اضغط ▶ → الموقع يعمل فوراً
5. **تجديد**: اضغط 🔄 → حدد نوع الاشتراك → يعمل فوراً

---

## للنشر على سيرفر حقيقي
غيّر في index.html:
```js
const _API = 'http://localhost:3000';
// إلى:
const _API = 'https://your-domain.com';
```
